# Paixão Pisos — Site Institucional

**Paixão Pisos** é uma empresa especializada em instalação e acabamento de pisos (Fulget, Granitina e Drenante) em Curitiba e região.  
Este repositório contém o código-fonte do site institucional da marca, desenvolvido para apresentar serviços, portfólio de trabalhos, catálogo de cores e canais de contato.

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3** (variáveis customizadas, grid, flexbox, animações)
- **JavaScript** (vanilla, sem dependências externas)
- **Integrações:** WhatsApp (link direto) e Instagram

## ✨ Funcionalidades

- **Design responsivo** — adaptado para mobile, tablet e desktop.
- **Menu fixo** com versão mobile (hamburger).
- **Hero section** com imagens de fundo e chamada para ação.
- **Seções:** Sobre, Serviços, Portfólio (resumido), Catálogo de Cores, Antes & Depois, Contato.
- **Filtro de portfólio** (Fulget, Granitina, Processo) com animações suaves.
- **Lightbox** para visualização ampliada das imagens.
- **Página exclusiva de acervo** (`portfolio.html`) com grade expansível para novos trabalhos.
- **Botões de WhatsApp e Instagram** fixos (flutuante + CTA).
- **Animações de revelação** durante a rolagem (ScrollReveal).
- **Links para orçamento gratuito** em todas as seções.
